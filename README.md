# TypePulse

TypePulse is a lightweight, privacy-friendly, fully local typing analytics tool.

## Features

- Track typing duration, key counts, and session counts
- Summarize by app and time range
- Visualize recent activity trends
- Export CSV for further analysis
- Local storage with one-click pause

## Development

- Install dependencies: `npm install`
- Run desktop app: `npm run tauri dev`

## Build

- Build frontend: `npm run build`
- Build desktop app: `npm run tauri build`
